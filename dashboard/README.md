# Aurika Enterprise Operations Dashboard
**Phase 13 Frontend Application**

## Overview
This directory contains the official enterprise frontend for Project Aurika, built with **React 19, TypeScript, Vite, TailwindCSS, Zustand, TanStack Query, Chart.js, and React Flow**.

## Running Locally
1. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
   The application will be accessible at `http://localhost:5173`.
3. Run unit tests:
   ```bash
   npm run test
   ```
4. Build production bundle:
   ```bash
   npm run build
   ```
